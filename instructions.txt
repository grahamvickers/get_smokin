You will need to run npm install in the front end and backend folders

Open Terminal run sudo mongod - this will start MongoDB

In mongoDB compass set up a DB named: db_get_smokin
Create a collection called students and import JSON data into collection

In a new terminal window , type cd and move to backend, npm run start
In a new terminal window, type cd and move to front-end, npm run serve

open browser window and navigate to: http://localhost:8003/
